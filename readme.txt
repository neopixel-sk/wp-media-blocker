Block media content (images, videos, audio) on your WordPress website until users choose to view them.

== Description ==

Media Blocker is a lightweight plugin that allows you to block various types of media content on your WordPress website. It creates an overlay on top of images, videos, and audio elements until a user clicks to view or play them.

**Key Features:**

* Block images, videos, and audio files
* Customizable overlay messages for each media type
* Simple settings page to configure the plugin
* Light footprint with minimal impact on page load times
* Works with standard WordPress content and embeds

= Use Cases =

* Enhance privacy by preventing automatic media loading
* Reduce bandwidth usage for users on metered connections
* Prevent autoplay of video/audio content
* Control what media content is immediately displayed to site visitors

== Installation ==

1. Upload the `media-blocker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Media Blocker to configure which types of media to block

== Frequently Asked Questions ==

= Which media types can be blocked? =

Currently, Media Blocker can block images, videos (including YouTube, Vimeo embeds), and audio elements.

= Will this work with my theme? =

Media Blocker works with most WordPress themes that use standard HTML for media elements.

= Does this plugin affect page speed? =

No, Media Blocker is designed to be lightweight and should not significantly impact your page loading times.
